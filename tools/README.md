# Tools (Development Only)

File-file di folder ini untuk development/maintenance. **Bukan bagian dari setup normal.**

| File | Fungsi |
|------|--------|
| `add.py` | Tambah akun CodeBuddy secara bulk |
| `bulk_harvest.py` | Harvest credential dari banyak akun |
| `bulk_harvest_v2.py` | Versi baru bulk harvest |
| `com.codebuddy2api.plist` | macOS LaunchDaemon (auto-start) |
| `setup_launchd.sh` | Setup auto-start di macOS |
